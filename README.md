# 💫 Deep Learning Indaba — Tutorials & Skills 2026 💫

Welcome to the **Tutorials & Skills 2026** repository. This is the home for all
tutorial notebooks and skills-session materials for Indaba 2026.

The repo is organised into two top-level folders:

| Folder | What goes here |
|:--|:--|
| [`practicals/`](practicals/) | Hands-on tutorial notebooks. One folder per tutorial. |
| [`skills/`](skills/) | Skills-session materials. One folder per session. |

Each **session leader owns a single folder** under one of these directories and
puts all of their material inside it. See **[CONTRIBUTING.md](CONTRIBUTING.md)**
for the exact folder convention and the contribution workflow.
